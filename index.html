Cheatsheet de Comandos

Referência rápida de comandos para DevOps e Segurança da Informação. Use as abas para navegar por categoria e a busca para filtrar qualquer comando.

ping hostVerifica conectividade com o host
ping -c 5 hostEnvia exatamente 5 pacotes
ping -i 0.2 hostIntervalo de 0.2s entre pacotes
ping -s 1472 hostDefine o tamanho do pacote em bytes
ping -t 64 hostDefine o TTL dos pacotes
ping6 hostPing via IPv6
